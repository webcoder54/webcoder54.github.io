What is Python?

Python is a high-level programming language used to store and use data. Python's easy to use syntax attracts many coders looking for a language.


Who created Python?

Python was created by Guido van Rossum in 1989.

Web Development

Python is used for back-end web development, the code on your server instead of the users' server.

Data Science

An example of data science is if you wanted to make a program that tells what the image shown is. You could say if it has light-brown hair it is a dog, or you could use machine learning. In simpler terms, that is where you give a bunch of pictures of a dog to a machine learning algorithm and the machine could recognize dogs.

Scripting

Scripting is writing programs to do tasks that you could do manually, but is easier with code.
